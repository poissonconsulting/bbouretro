# Plot population growth distributions

Create histograms of simulated population growth (\\\lambda\\) values.

## Usage

``` r
bbr_plot_growth_distributions(growth)
```

## Arguments

- growth:

  A data frame generated by
  [`bbr_growth()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_growth.md).

## Value

A ggplot object.

## Details

Plots are generated that show the distribution of simulated population
growth (\\\lambda\\) values, the mean estimate (red line). In addition,
a hashed line indicates where \\\lambda = 1\\. Plots allow users to
evaluate the symmetry of the distributions of \\\lambda\\.

## Examples

``` r
if (FALSE) { # \dontrun{
recruitment_est <- bbr_recruitment(bboudata::bbourecruit_a)
survival_est <- bbr_survival(bboudata::bbousurv_a)
growth_est <- bbr_growth(survival_est, recruitment_est)

bbr_plot_growth_distributions(growth_est)
} # }
```
