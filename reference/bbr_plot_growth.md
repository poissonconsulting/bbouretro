# Plot population growth

A plot of population growth (\\\lambda\\) estimates is given for the
population unit.

## Usage

``` r
bbr_plot_growth(growth)
```

## Arguments

- growth:

  A data frame generated by
  [`bbr_growth()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_growth.md).

## Value

A ggplot object.

## Examples

``` r
if (FALSE) { # \dontrun{
recruitment_est <- bbr_recruitment(bboudata::bbourecruit_a)
survival_est <- bbr_survival(bboudata::bbousurv_a)
growth_est <- bbr_growth(survival_est, recruitment_est)

bbr_plot_growth(growth_est)
} # }
```
