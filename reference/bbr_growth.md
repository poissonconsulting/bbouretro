# Simulate population growth

This function uses the output of
[`bbr_survival()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_survival.md)
and
[`bbr_recruitment()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_recruitment.md)
to estimate population growth (\\\lambda\\) using the Hatter-Bergerud
equation (Hatter and Bergerud, 1991). Monte Carlo simulation is used to
generate confidence limits.

## Usage

``` r
bbr_growth(survival, recruitment)
```

## Arguments

- survival:

  A data frame generated by
  [`bbr_survival()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_survival.md).

- recruitment:

  A data frame generated by
  [`bbr_recruitment()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_recruitment.md).

## Value

A data.frame.

## Details

See the `vignette("methods", package = "bbouretro")` for descriptions of
the equations used. The raw_values can be plotted using
[`bbr_plot_growth_distributions()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_plot_growth_distributions.md)
and the summary data frame can be output using
[`bbr_growth_summarize()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_growth_summarize.md)
or plotted using
[`bbr_plot_growth()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_plot_growth.md).

## References

Hatter, Ian, and Wendy Bergerud. 1991. “Moose Recruitment, Adult
Mortality and Rate of Change” 27: 65–73.

## Examples

``` r
if (FALSE) { # \dontrun{
recruitment_est <- bbr_recruitment(bboudata::bbourecruit_a)
survival_est <- bbr_survival(bboudata::bbousurv_a)

growth_est <- bbr_growth(survival_est, recruitment_est)
} # }
```
