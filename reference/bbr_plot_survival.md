# Plot survival

A plot of yearly survival is given for each population unit.

## Usage

``` r
bbr_plot_survival(survival)
```

## Arguments

- survival:

  A data frame generated by
  [`bbr_survival()`](https://poissonconsulting.github.io/bbouretro/reference/bbr_survival.md).

## Value

A ggplot object.

## Examples

``` r
if (FALSE) { # \dontrun{
survival_est <- bbr_survival(bboudata::bbousurv_a)

bbr_plot_survival(survival_est)
} # }
```
